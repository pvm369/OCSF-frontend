Data Pipeline

HOW IT WORKS

Each raw event passes through a 5-stage pipeline. The engine handles detection and routing automatically — your only job is the YAML.

01
📄
Raw JSON Logs
JSONL input from any supported security source
02
🔍
Auto-Detect
Matches required fields to identify log source
03
📋
Load YAML Rules
Reads mapping file for the detected source
04
⚡
Route + Resolve
Templates, enums, wildcards resolved per event
05
✅
OCSF Output
Normalized JSON + YAML ready for your pipeline
